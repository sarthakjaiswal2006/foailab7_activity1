# 🚀 Public API Playground

A modern, interactive web application that explores multiple public APIs in one place.
Built with a clean UI, smooth animations, and dynamic data fetching using JavaScript.

## 📌 Features

### 🐶 Dog Finder

* Fetch a random dog image
* Automatically extract and display breed name
* Copy image URL with one click

### 😂 Joke Generator

* Get random jokes instantly
* Displays setup + punchline
* "Next Joke" button for continuous fun

### 👤 Random User Profile

* Generate real-looking user data
* Displays name, profile image, email, country
* Includes additional info like age

---

## 🎯 Advanced UI Features

* 🔥 Modern glassmorphism design
* 🎨 Dark gradient theme with glow effects
* ✨ Smooth animations and transitions
* 📱 Fully responsive layout
* 🧭 Navigation-based sections (Home, Dog, Joke, User APIs)
* ⚡ Interactive buttons and hover effects

---

## 🛠️ Tech Stack

* **HTML5**
* **CSS3 (Modern UI + Animations)**
* **JavaScript (ES6+)**
* **Fetch API & Async/Await**

## 🔗 APIs Used

* 🐶 Dog API → https://dog.ceo/api/breeds/image/random
* 😂 Joke API → https://official-joke-api.appspot.com/random_joke
* 👤 Random User API → https://randomuser.me/api/
